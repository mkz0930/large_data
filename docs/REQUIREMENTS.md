# 需求文档

## 项目目标

大批量 Amazon ASIN 抓取器，支持关键词搜索和分类扩展抓取。

## 功能需求

### 1. 单关键词抓取

- 输入关键词，使用 ScraperAPI 抓取 Amazon 搜索结果
- 智能停止：当产品销量低于阈值时自动停止
- 获取 ASIN 分类数据（通过卖家精灵）
- 分析产品分类分布
- 扩展抓取热门分类的更多 ASIN

### 2. 批量关键词抓取

- 从文件读取多个关键词
- 依次执行单关键词抓取流程
- 汇总所有抓取结果

### 3. 数据存储

- 使用 SQLite 数据库存储
- 支持增量抓取（去重）
- 记录抓取任务状态

## 非功能需求

### 性能

- 支持并发请求（默认 10 并发）
- 请求超时 60 秒
- 失败重试 5 次

### 可配置性

- 国家代码
- 最大页数
- 销量阈值
- 热门分类数量
- 数据库路径

## 依赖

- ScraperAPI：Amazon 数据抓取
- 卖家精灵（可选）：ASIN 分类数据
